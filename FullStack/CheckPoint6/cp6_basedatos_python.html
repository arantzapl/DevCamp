¿Qué es MongoDB?

Es una base de datos NoSQL que se orienta a documentos. Está creada para guardar datos de forma flexible. A diferencia de las bases de datos relacionales (SQL) que almacenan sus datos en filas y tablas, MongoDB almacena los datos en un formato similar a JSON. Lo que permite que sea una estructura más libre, ideal para aplicaciones que necesiten cambios en el esquema de datos.

Características

  • Está orientada a objetos. Los datos se almacenan en una extensión de JSON, lo que permite que sean más complejos que los modelos tradicionales.
  • MongoDB está diseñado para permitir distribuir datos mediante múltiples servidores; lo que es ideal para aplicaciones que manejen un gran volumen de datos y alta disponibilidad de ellos.
  • Admite consultas ad-hoc, búsquedas por campo, consultas de rango y expresiones regulares; además de frameworks que permiten realizar operaciones como sumas, promedios y agrupaciones de datos.
  • Permite la creación de índices en cualquier campo, lo que mejora la eficiencia de las consultas.

Cuándo usar MongoDB

  • En aplicaciones con datos en evolución y grandes tasas de inserciones.
  • En aplicaciones con esquemas de datos no estructurados.
  • En aplicaciones en tiempo real que necesitan una iteración y despliegue rápidos.
